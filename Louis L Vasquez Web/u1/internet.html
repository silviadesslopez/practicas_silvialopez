¿Qué es internet?

Es una red de computadoras que se encuentran interconectadas a nivel mundial para compartit información por medio de diversos protocolos y servicios. Los protocolos permiten la unión en la red y la transmisión de información, fragmentan, unen y traducen los mensajes para que los datos lleguen correctamente de una computadora a otra. Los servicios permiten a los usuarios crear, enviar, recibir y acceder a la información.


Tipos de proovedores de internet

Un Proveedor de Servicio de Servicios de Internet, o ISP, son empresas que proporcionan el acceso a internet a través de una conexión de DSL o banda ancha. También suelen ofrexer servicios relacionados como cuentas de correo electrónico, hospedaje web, registro de nombres de dominio y servicios telefónicos.


¿Qué son los protocolos de aplicación?

Es el conjunto de reglas y normas que permiten que los dispositivos y las aplicaciones puedan comunicarse entre sí de manera eficiente y efectiva. Define cómo se deben estructurar los mensajes, cómo se deben envuiar y recibir y que significado tienen los diferentes tipos de mensaje. Es esencial en la comunicación entre dispositivos y aplicaciones.


Concepto de interfaz de usuario

La interfaz de usuario, o UI, es el punto de interacción y comunicación humano-computadora en un dispositivo, incluye pantallas de visualización, teclados, un mouse y la apariencia de un escritorio. Es la forma en que un usuario interactúa con una aplicación o un sitio web. Generalmente son espacios intuitivos, de amigable utilización, que a través de distintas dinámicas permiten al usuario manejar los complejos procesos del sistema informático.


Concepto de experiencia de usuario

La experiencia de usuario, o UX, se refiere a la forma en que una persona se siente al interactuar con un producto, servicio o plataforma digital. Esta experiencia abarca todos los aspectos de interacción, desde la navegación y la apariencia visual, hasta la facilidad de uso y la satisfacción del usuario. Se trata en cómo se siente un usuario al usar un sitio web o aplicación móvil.


¿Qué es WWW?

La World Wide Web, o WWW, es una red de alcance mundial, que consiste en un complejo sistema de hipertextos e hipermedios interconectados entre sí y a los que puede tenerse acceso mediante una conexión a Internet y un conjunto de software especializado. Para poder visitarla es necesario de un navegador o explorador, se respresenta en una serie de ventanas dotadas de texto y multimedios de cada página web visitada.


¿Qué son los nombres de dominio DNS?

Un nombre de dominio localiza a una organización u otra entidada en internet, es el sistema de nombres amigable para dar direcciones a servidores y páginas web. le proporciona a cada servidor, una dirección única, fácil de recordar y de deletrear. Simultáneamente, los nombres de dominio ocultan la dirección IP técnica en la que la mayoría de los visitantes no están interesados.

Son conformadas por: Dominio de nivel superior ".com", Nombre de dominio "sitio nuevo", o cualquier nombre que desees otorgarle, y el prefijo "www" que hace referencia al servidor.


¿Qué es hosting?

El hosting, o alojamiento web, es un servicio digital que tiene como función almacenar la información y contenido de un sitio web en centros de procesamiento de datos seguros y veloces. Es esencial para hacer que un sitio web esté disponible en internet, ya que brinda el espacio y los recursos necesarios para almacenarlo. Cuando alguien visita el sitio web, el navegador se conecta al servidor de hosting donde este alojado y descarga los archivos necesarios para mostrar el contenido en el navegador del usuario.

Se caracterizan por;

  • Conectividad: Para acceder a la información es necesario contar con conexión a internet.
  • Almacenamiento: La información de los sitios web se resguardan en grandes bases que centralizan la información de una gran cantidad de dominios.
  • Capacidad: Los servidores hosting requieren una amplia disponibilidad de espacio y una infraestructura especializada, y son una opción para mantener segura una página y lista para ser ejecutada.
  • Integración: Estos servicios generalmente incluyen la contratación de un dominio web y de una plataforma de correo electrónico.
  • Accesibilidad: Al resguardar la información de tu página en estos servidores tienes la certeza de que no la perderás si algo le ocurre a tu equipo.
